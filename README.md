  制作テーマ１：福岡の観光名所
  　概要：

  目的：福岡県の魅力や観光地の良さを多くの人に知ってもらうため。福岡県には歴史的な建造物や自然豊かな景観、グルメスポットなど様々な魅力があるが、県外の人には十分に知られていない場所も多い。本制作を通して福岡県の観光資源を紹介し、観光への興味を持ってもらうこと

  制作テーマ２：世界の名所
　　概要：近年、海外旅行の情報収集はインターネットを利用することが一般的となっていが、多くの観光サイトは情報量が多く、初めて海外旅行を計画する人にとっては必要な情報を見つけにくい場合がある。そこで、世界各国の名所について、歴史や見どころ、アクセス方法などを分かりやすくまとめたWebサイトを制作し、利用者が旅行先を比較・検討しやすいサイトを目指す。
